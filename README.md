## How to run

Frontend and backend are in separate directories. You can run them separately.
Frontend is built with [Remix](https://remix.run/)

```bash
nvm use
cd fe
yarn dev
```

Open one more terminal and run the backend with the following commands:

```bash
nvm use
cd ws
yarn dev
```
